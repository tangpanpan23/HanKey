## 🚀 快速开始

### 1. 环境要求
- Go 1.18+
- 现代浏览器（支持ES6+）

### 2. 启动后端服务
```bash
# 进入API目录
cd app/hanbao/api

# 启动服务
go run hanbao.go -f etc/hanbao-api.yaml
```

服务将在 `http://localhost:8080` 启动

### 3. 打开演示页面
```bash
# 在浏览器中打开
open web/index.html
```

或者使用本地服务器：
```bash
cd web
python3 -m http.server 3000
# 访问 http://localhost:3000
```
